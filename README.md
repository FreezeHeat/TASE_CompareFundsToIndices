# TASE_CompareFundsToIndices
Compare TASE trust funds, ETFs and foreign ETFs to their indices according to their return (Net or Gross) to find hidden costs

## Data
Data includes sources, indices performance is compared to a fund's performance, exposing hidden costs.

The data is in Hebrew as it's based on TASE.

## Requirements
To edit the data you need Libreoffice Calc or any other ods compatible software (open source format)

Otherwise, any PDF reader would be fine with the PDF file
